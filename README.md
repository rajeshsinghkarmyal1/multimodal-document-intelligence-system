# Multimodal Document Intelligence System

## Overview

AI-powered document processing platform that extracts text from images and scanned documents using OCR and generates intelligent summaries using OpenAI.

## Features

- OCR Text Extraction
- Document Summarization
- Question Answering
- FastAPI Backend
- OpenAI Integration

## Tech Stack

- Python
- FastAPI
- OpenAI API
- Tesseract OCR
- Pillow
